## Switch

The `now switch` command is used to switch to a different team scope when logged in with <ProductNameShort /> CLI. You can choose to select a team from a list of all those you are part of or specify a team when entering the command.

### Basic Usage

<Snippet dark text="now switch" />
<Caption>Using the <InlineCode>now switch</InlineCode> command to change team scope with <ProductNameShort /> CLI.</Caption>

### Extended Usage

<Snippet dark text="now switch [team-name]" />
<Caption>Using the <InlineCode>now switch</InlineCode> command to change to a specific team scope with <ProductNameShort /> CLI.</Caption>

### Global Options

The following [global options](#options) can be passed when using the `now switch` command:

- `--debug`
- `--help`

For more information on global options and their usage, refer to the [options section](#options).
